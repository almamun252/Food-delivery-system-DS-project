# Online Food Delivery System (C)

This project implements a console-based food delivery system using C.

## Features

- Restaurant management
- Food menu system
- Order placement
- Graph-based shortest delivery path (Dijkstra)
- Nearest rider assignment
- Order history
- Admin panel

## Data Structures Used

- Graph
- Arrays
- File handling

## Files

locations.txt – delivery locations  
graph.txt – distance between locations  
riders.txt – rider data  
data.txt – restaurant & dish data
